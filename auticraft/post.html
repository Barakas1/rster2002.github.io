Maak een post

Gebruik '<p>' om een paragraaf te maken en '<h1>' om een title of kopje te maken. Voorbeeld: '<h1>Mijn mooie bouwwerk</h1><p>Ik ben de laatste tijd bezig geweest met een bouwwerk.</p>' Je kan ook gewoonen javascript gebruiken. Gebruik alleen geen 'script', of 'link' want als dat gebeurd dan wordt de post verwijderd en krijg je een waarschuwing. Als je het daarna nog eens doet wordt je op de black-list gezet.

Type hieronder het de titel van je artikel.

Type hieronder een url naar een foto

Type hieronder het soort artikel. Dat kan zijn: update, mededeling, event of wedstrijd.
