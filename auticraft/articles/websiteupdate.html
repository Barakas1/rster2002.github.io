Website updates

De website in het algemeen

Ik zal de website vaker proberen te updaten met challenges en content. De website gaat zeker nog niet weg, maar er worden wel veel dingen veranderd.

Dingen die zijn veranderd

  • De homepage is geupdate met een nieuw design

  • Betere mobiele ondersteuning

  • De map is van de website gehaalt om te updaten

Dingen die gaan veranderen

  • Er komt een challenges pagina waar je terecht kan als je je verveelt op de server

  • De map die op de website staat zal worden deupdate zodat het makkelijker wordt om te vinden waar je naartoe moet

  • De shops pagina wordt of verwijderd of deupdate

  • Misschien ga ik maken dat je zelf dingen kan posten op de website zodat je sneller een project of event kan aankondigen

  • Dat is het z'n beetje!
